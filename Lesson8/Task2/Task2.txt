Как я думаю, в данном примере имеется ввиду, что если "a" не расположен 
в верхнем уровне, а именно в Window, то "а" будет равен 1, иначе - undefined,
поэтому в сообщении будет undefined:

if (!("a" in window)) {
    var a = 1;
}
alert(a);



В данном примере получается, что функция a(x) - это тоже самое,
что a = function a(x), функция будет некорректной:

var b = function a(x) {
    x && a(--x);
};
alert(a);



Здесь будет выдано тело функции, никакого аргумента в данную функцию передан не был,
а если запросить alert(a(2)), то результам будет 4:

function a(x) {
    return x * 2;
}
var a;
alert(a);

function b(x, y, a) {
